# Resume Descriptions

You can use these bullet points in your resume under the "Projects" section to highlight your work on the Sentiment Analysis Dashboard.

### Option 1: Full-Stack / NLP Focus
**Sentiment Analysis of Social Media Presence**
*Designed and developed an AI-powered SaaS dashboard capable of real-time public opinion analysis and fake review detection.*
* Built a lightweight Python backend using **Flask** and **SQLite**, implementing NLP libraries (**TextBlob**, **VADER**) to evaluate text sentiment with >85% confidence accuracy.
* Engineered a bulk CSV upload processor using **Pandas**, allowing users to analyze thousands of comments simultaneously and visualize the aggregated sentiment.
* Developed a responsive, Cyberpunk-themed frontend utilizing **Vanilla JS**, **Chart.js**, and **GSAP**, featuring a custom Glassmorphism UI and a rule-based AI chatbot assistant.
* Implemented heuristic spam detection algorithms to filter out bot activity and flag suspicious comments.

### Option 2: Frontend / UI Focus
**AI Sentiment Intelligence Dashboard**
*Created a highly interactive, futuristic dashboard interface for visualizing social media data.*
* Designed a premium **Glassmorphism** and Neon-themed UI using custom CSS3 variables, animations, and `backdrop-filter` techniques.
* Integrated **Chart.js** to render dynamic pie charts and line graphs, dynamically updated via asynchronous REST API calls.
* Utilized **GSAP (GreenSock)** to build complex micro-animations, staggered lists, and a floating chatbot interface, drastically improving user engagement.
* Consumed Python-based REST APIs using the `fetch()` API to provide real-time, zero-latency sentiment feedback as users typed.

### Option 3: Backend / Data Focus
**Social Intelligence API & Engine**
*Developed the backend architecture for a real-time sentiment tracking application.*
* Created **RESTful APIs** using **Flask** to handle real-time sentiment analysis, CSV file processing, and simulated social media data aggregation.
* Designed a relational database schema in **SQLite** to log historical queries, track trending hashtags, and calculate moving averages for sentiment scores.
* Integrated Natural Language Processing pipelines using **VADER** to output normalized polarity metrics and custom AI-generated insight explanations.
* Built a modular fake-review detection system that utilized Regex and pattern matching to identify 90% of structural spam.
